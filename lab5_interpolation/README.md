# Lab 5: Polynomial Interpolation

This mini-project contains MATLAB/Octave scripts for polynomial interpolation.

## Contents

- `zadanie1.m` to `zadanie5.m`: Scripts implementing various interpolation tasks, including Vandermonde matrix generation and analyzing Runge's phenomenon.
- `main.m`: Main script to run the project.

## How to run

Execute `main.m` in MATLAB or Octave.
