# Lab 4: Solving Non-linear Equations

This mini-project contains MATLAB/Octave scripts for solving non-linear equations using numerical methods.

## Contents

- `bisection_method.m`: Implementation of the Bisection method.
- `secant_method.m`: Implementation of the Secant method.
- `impedance_magnitude.m`: Function calculating impedance magnitude.
- `rocket_velocity.m`: Function calculating rocket velocity.
- `estimate_execution_time.m`: Script to estimate execution time.
- `main.m`: Main script to run the project.

## How to run

Execute `main.m` in MATLAB or Octave.
