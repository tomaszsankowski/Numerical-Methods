# Lab 3: Solving Systems of Linear Equations

This mini-project contains MATLAB/Octave scripts for solving systems of linear equations using various numerical methods.

## Contents

- `solve_direct.m`: Direct method for solving linear equations.
- `solve_Jacobi.m`: Iterative Jacobi method.
- `solve_Gauss_Seidel.m`: Iterative Gauss-Seidel method.
- `generate_matrix.m`: Generates test matrices.
- `plot_direct.m`, `plot_problem_5.m`: Visualization scripts.
- `filtr_dielektryczny.mat`: Data file containing a dielectric filter matrix.
- `main.m`: Main script to run the project.

## How to run

Execute `main.m` in MATLAB or Octave.
