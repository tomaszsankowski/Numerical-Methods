# Lab 2: PageRank and Circle Generation

This mini-project contains MATLAB/Octave scripts that implement the PageRank algorithm and generate/plot circles.

## Contents

- `page_rank.m`: Implementation of the PageRank algorithm.
- `generate_circles.m`: Generates random circles.
- `plot_circle.m`, `plot_circles.m`, `plot_circle_areas.m`: Functions for visualizing circles and their areas.
- `plot_counts_mean.m`, `plot_PageRank.m`: Visualization scripts for PageRank results.
- `main.m`: Main script to run the project.

## How to run

Execute `main.m` in MATLAB or Octave.
