# Project 1: MACD Stock Index

This mini-project contains a Python script to calculate and analyze the MACD (Moving Average Convergence Divergence) stock market indicator.

## Contents

- `main.py`: Main Python script that calculates the MACD indicator and plots the results.
- `pkn.csv`, `sp500.csv`: Historical stock market data used for the analysis.

## How to run

1. Ensure you have Python installed.
2. Install required dependencies (e.g., `pandas`, `matplotlib`).
3. Run `python main.py`.
