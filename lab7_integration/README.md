# Lab 7: Numerical Integration

This mini-project contains MATLAB/Octave scripts for numerical integration.

## Contents

- `zadanie1.m` to `zadanie5.m`: Scripts implementing numerical integration methods, such as the rectangular method, and calculating integration errors.
- `get_lake_depth.m`: Function to get lake depth data.
- `main.m`: Main script to run the project.

## How to run

Execute `main.m` in MATLAB or Octave.
