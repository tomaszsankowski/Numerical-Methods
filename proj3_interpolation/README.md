# Project 3: Interpolation

This mini-project contains Python scripts for advanced interpolation techniques applied to geographical paths.

## Contents

- `main.py`: Main Python script to run the interpolation algorithms.
- `metoda_wielomianowa.py`: Implementation of polynomial interpolation.
- `paths/`: Directory containing CSV files with geographical path data (e.g., `MountEverest.csv`, `WielkiKanionKolorado.csv`).
- `plots/`: Directory for saving generated plots.

## How to run

1. Ensure you have Python installed.
2. Install required dependencies (e.g., `numpy`, `matplotlib`, `pandas`).
3. Run `python main.py`.
